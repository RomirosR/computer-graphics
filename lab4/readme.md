## Формат входного файла

```
n
X1_1 Y1_1 X2_1 Y2_1
X1_2 Y1_2 X2_2 Y2_2
...
X1_n Y1_n X2_n Y2_n
Xmin Ymin Xmax Ymax
```

* `n` — количество отрезков.
* `X1_i Y1_i X2_i Y2_i` — координаты концов каждого отрезка.
* `Xmin Ymin Xmax Ymax` — координаты прямоугольного окна (для Лианга-Барски) или вершин многоугольника (для отсечения по выпуклому многоугольнику).

Пример файла:

```
5
10 10 100 50
50 20 150 80
30 70 120 30
80 10 180 60
0 40 200 40
20 20 150 70
```

---

## Алгоритмы

### 1. Лианг-Барски

* Отрезок задаётся параметрически: `x = x1 + t*(x2-x1), y = y1 + t*(y2-y1), t∈[0,1]`.
* Вычисляются значения `tE` и `tL`, определяющие видимую часть отрезка.
* Эффект: точное отсечение отрезков прямоугольным окном.

### 2. Отсечение отрезков выпуклым многоугольником (Cyrus-Beck)

* Используется параметрическое представление отрезка.
* Для каждой границы многоугольника вычисляются значения входа (`tE`) и выхода (`tL`).
* Видимая часть отрезка: `tE ≤ t ≤ tL`.
* Эффект: корректное отсечение для произвольного выпуклого многоугольника.
