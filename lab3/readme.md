## Алгоритмы

### 1. Пошаговый алгоритм

* Идея: пошаговое вычисление координат по шагам вдоль X или Y.
* Особенности: простая реализация, подходит для обучения.
* Эффект: линия рисуется «наивно», с округлением координат.

### 2. Алгоритм ЦДА

* Идея: вычисление приращений `dx/n` и `dy/n` и последовательное добавление к начальной точке.
* Особенности: более точное распределение пикселей по сравнению с пошаговым.
* Эффект: линия получается непрерывной, равномерной.

### 3. Алгоритм Брезенхема (линия)

* Идея: целочисленные вычисления ошибок для выбора следующего пикселя.
* Особенности: эффективный, не использует дробные значения.
* Эффект: «идеальные» линии без дробных координат, быстрый алгоритм.

### 4. Алгоритм Брезенхема (окружность)

* Идея: рисует пиксели по симметрии окружности, используя целые числа.
* Особенности: каждый шаг рисует точки в 8 октантах.
* Эффект: точная окружность с минимальными вычислениями.
